Mova os 2 arquivos para a pasta superior e substitua.
Caso você não tenha instalado o resource mysql-async: https://brouznouf.github.io/fivem-mysql-async/

Para adicionar as permissões utilize o sistema ACE, em seu server.cfg adicione igual o exemplo abaixo

add_ace identifier.steam:<STEAMID> "flux.bypass" allow
add_ace identifier.steam:<STEAMID> "flux.admin" allow

Vai ficar similar aos de baixo:
add_ace identifier.steam:100000000000000 "flux.bypass" allow
add_ace identifier.steam:100000000000000 "flux.admin" allow
